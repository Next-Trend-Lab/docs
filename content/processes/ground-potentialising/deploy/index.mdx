For each event prisma organises, we aim to set up [minimal viable infrastructure](/patterns/minimal%20viable%20infrastructure.md) to amplify the facilitation capabilities. This is done by deploying to partner [hubs](/collaborators/communities-of-place). From there, the facilitation capabilities are extended to all teams formed during the event. 

In the lead up to [CATS](/events/summit.md), we are automating this process so that hubs can deploy this infrastructure themselves. Deployment authorisation will be managed through the issuance of hub NFTs.
